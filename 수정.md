# 📈 [Prompt] Streamlit 대시보드(app.py) 실시간 Raw 신호 파형 시각화 추가 지침

현재 정합 중인 `app.py` 관제 대시보드 화면에, PRD 명세 규격인 `(N, 1, 2000)` 가변 길이 시계열 신호를 시각적으로 증명할 수 있는 **[실시간 신호 타임시리즈 청크 스트림 차트]** 컴포넌트를 추가해줘.

기존 다크모드 CSS 테마와 컴포넌트 간의 레이아웃 배치를 해치지 않도록 아래 요건에 맞춰 정밀하게 코드를 확장해줘.

---

## 1. 시각화 컴포넌트 추가 및 레이아웃 배치 요건
- **위치 선정:** 대시보드 메인 영역의 `left_col` 내부, 즉 **기존 [SHAP 워터폴 기여도 분석 차트] 바로 아랫단**에 나란히 붙도록 설계해줘.
- **디자인 통일:** 차트 배경색은 배경 대륙과 일치하는 `#1E293B`로 채우고, 그리드 선 색상은 `#334155`로 지정하여 다크모드 대시보드의 몰입감을 유지해줘.

## 2. 데이터 연동 및 파형 생성 로직 (`(N, 1, 2000)` 규격 증빙)
- 사용자가 특정 CSV 샘플을 선택하여 `InferenceRouter` 추론 엔진이 자동 가동될 때, 연산에 주입된 **2,000개 행짜리 1차원 `rms` 시계열 데이터**를 소스로 받아 그래프를 그려야 해.
- 만약 실제 런타임 텐서 데이터를 어플리케이션단에서 직접 슬라이싱하기 어려운 상황을 대비해, 현재 대시보드 상단 레이아웃의 데이터 특징값(`_mean`, `_var`)을 기반으로 **2,000개의 타임스탬프를 가진 동적 가인(Sine/Noise 혼합) 파형 스트림**을 즉석에서 생성하는 방어 코드도 내장해줘.
  - *전류(Current_U_var), 모터진동(Vib_Motor_var), 감속기진동(Vib_TM_var) 중 사용자가 우측 라디오 버튼에서 선택한 [타겟 고장 유형]과 매핑되는 핵심 센서 신호의 파형이 표출되도록 스마트 라우팅해줘.*

## 3. 차트 미장센 및 폰트 깨짐 완벽 예방 (Matplotlib 명세)
- 맥북 환경에서 차트가 로드될 때 오류가 없도록 글꼴을 명확히 제어해야 해.
- 차트 제목, X축(Time Index / 2000 samples), Y축(Signal Amplitude) 레이블의 색상은 `#94A3B8`로 지정하고 폰트는 `'Apple SD Gothic Neo'`로 고정해줘.
- 신호 파형 라인의 색상은 가독성이 극대화되도록 일렉트릭 블루 계열인 **`#38BDF8`**로 렌더링해줘.
